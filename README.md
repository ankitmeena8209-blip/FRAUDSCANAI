# FraudScanAI

A premium, single-page cybersecurity SaaS UI for analyzing suspicious content such as fake news, scam messages, phishing links, fraud emails, and screenshots.

## Tech stack
- React 19
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Hook Form
- Zod
- Recharts
- LocalStorage

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to Vercel
1. Push this repository to GitHub.
2. Import the repository in Vercel.
3. Vercel will use the existing Vite build settings automatically.
